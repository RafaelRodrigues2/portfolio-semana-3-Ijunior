# Portfólio Pessoal – Rafael Rodrigues

## Sobre o projeto

Este projeto consiste no desenvolvimento de um portfólio pessoal utilizando HTML e CSS, criado como parte das atividades da iJunior.

O objetivo foi construir uma página de apresentação optei por demonstrar minha formação acadêmica, experiência profissional, habilidades técnicas e formas de contato.

Como diferencial, o design foi inspirado em uma estética **neon/cyberpunk**, utilizando uma paleta baseada em tons de roxo, rosa e azul escuro.

## Funcionalidades

* Apresentação pessoal com foto e descrição.
* Formação acadêmica.
* Experiência profissional.
* Habilidades técnicas organizadas por categoria.
* Idiomas.
* Área de contatos com links para GitHub e e-mail.
* Interface responsiva construída com Flexbox.
* Efeitos visuais utilizando sombras, animações e transições em CSS.
* Componentes interativos com efeitos de hover.

## Tecnologias utilizadas

* HTML5
* CSS3
* Flexbox
* Git
* GitHub

## Organização do projeto

Durante o desenvolvimento foi utilizado um fluxo de versionamento baseado em branches:

* `main` – versão estável do projeto.
* `homologacao` – ambiente de validação.
* `develop` – integração das funcionalidades.
* `coding/...` – desenvolvimento das novas funcionalidades.

Esse fluxo permitiu manter o histórico organizado e simular um ambiente de desenvolvimento colaborativo.

## Objetivo de aprendizado

Além da construção do portfólio, o projeto teve como foco o aprendizado de:

* Estruturação semântica com HTML.
* Layouts utilizando Flexbox.
* Estilização avançada com CSS.
* Organização de código.
* Boas práticas de versionamento com Git e GitHub.
